\# Persona Multiplier Workflow



\## Goal

Transform one recruiter story into multiple outreach styles using AI personas.



\---



\## Inputs

\- Original recruiter story

\- Target person or company



\---



\## Workflow



1\. User enters a recruiter story

2\. User enters a target company or recipient

3\. Streamlit sends prompts to OpenAI API

4\. AI generates:

&#x20;  - Direct Headhunter version

&#x20;  - Empathetic Coach version

&#x20;  - Trusted Advisor version

5\. Each version includes contextual personalization

6\. Recruiter copies the preferred version for LinkedIn outreach



\---



\## Technology Stack

\- Python

\- Streamlit

\- OpenAI API

\- Prompt Engineering



\---



\## Commercial Value

Recruiters often struggle to adapt outreach tone across different audiences.



This tool rapidly generates multiple commercially relevant messaging styles while maintaining the same core story and adding contextual personalization.



This reduces generic outreach and improves engagement quality at scale.
