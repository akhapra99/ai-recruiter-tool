# Persona Multiplier

AI-powered recruiter outreach tool that transforms one recruiter story into multiple communication personas while maintaining the same core message.

## Features
- Multi-persona AI rewriting
- Contextual personalization
- Recruiter-focused outreach generation
- LinkedIn-ready outputs

## Personas
- Direct Headhunter
- Empathetic Coach
- Trusted Advisor

## Tech Stack
- Python
- Streamlit
- OpenAI API

## Setup

1. Install dependencies:
pip install -r requirements.txt

2. Create a `.env` file and add:
OPENAI_API_KEY=your_api_key_here

3. Run the application:
streamlit run app.py

## Project Goal
The goal of this prototype is to help recruiters adapt outreach communication styles more effectively using AI orchestration and contextual personalization.
